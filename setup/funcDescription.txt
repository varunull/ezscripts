
<ff>
<shortInfo>find files</shortInfo>
<desc>This function internally used the find tool to find for files recursively in the current directory</desc>
<usages>
<cmd>ff rockyou.txt</cmd><cmdPurp>Find the file rockyou.txt recursively in all the folders of the current directory</cmdPurp>
</usages>
</ff>

<slc>
<shortInfo>show last commands</shortInfo>
<desc>Utility to check the most recent usages of a particular tool. Particularly saves the manual work of parsing through the history to find the usages. The number of commands shown in the output can be controlled via the arguments.</desc>
<usages>
<cmd>slc grep 3</cmd><cmdPurp>Show the latest three usages of grep command</cmdPurp>
<cmd>slc find</cmd><cmdPurp>Show all the usages of find command that are present in the bash_history</cmdPurp>
</usages>
</slc>

<ffg>
<shortInfo>find files using grep</shortInfo>
<desc>This function creates a wildcard string using the arguments, and then the resultant string is used to search for the files recursively in the current directory</desc>
<usages>
<cmd>ffg lema jack</cmd><cmdPurp>Find the file names with pattern *lema*jack* recursively in the current directory</cmdPurp>
</usages>
</ffg>

<findcmdhelp>
<shortInfo>find command help</shortInfo>
<desc>This function takes a tool name as the argument and lists down all the examples of usages of it. These usages were stored to and file using the related command 'addcmdhelp'</desc>
<usages>
<cmd>findcmdhelp nmap</cmd><cmdPurp>Find all the saved command usages for the nmap. These were saved using the addcmdhelp function</cmdPurp>
</usages>
</findcmdhelp>

<addcmdhelp>
<shortInfo>add command help</shortInfo>
<desc>This function is used to save the commands for different tools that we use frequently. It takes the tool name and the corresponding command as the argument, which can then be quickly retrieved using 'findcmdhelp' command.</desc>
<usages>
<cmd>addcmdhelp nmap 'sudo nmap -sV --version-all 10.10.102.93'</cmd><cmdPurp>Add this command usage for nmap which can later be retrieved by the 'findcmdhelp' command.</cmdPurp>
</usages>
</addcmdhelp>
